# Lamma+ Branding Guidelines

## Core Identity

**Name:** Lamma+ (The Gathering)
**Symbol:** The Date Tree (representing shelter, sustenance, and gathering).

## Color Palette

### Primary Colors

- **Deep Teal (`#0D7377`)**: Represents the Oasis, depth, and night. Used for headers, primary backgrounds in light mode, and heavy accents.
- **Warm Gold (`#F5B820`)**: Represents the Dates, desert sun, and warmth. Used for Primary Actions (Follow, Subscribe), highlights, and active states.

### Secondary Colors

- **Teal Deep (`#1D4E5F`)**: Darker shade for gradients/borders.
- **Teal Light (`#E6F4F4`)**: Soft background for banners/cards (Light Mode).
- **Gold Light (`#FEF9E7`)**: Soft background for historical/special items.

### Neutrals

- **Slate 900 (`#0f172a`)**: Main background for Dark Mode (Night Theme).
- **Slate 800 (`#1e293b`)**: Card/surfaces background in Dark Mode.
- **White/Off-white**: Text and clean surfaces.

## Typography

- **Headings**: Bold, distinctive (Inter/System).
- **Body**: Clean, readable.

## Components

- **Buttons**:
  - **Primary**: Gold background, Dark text (High visibility).
  - **Secondary/Following**: Slate/Dark background, White text.
  - **Ghost/Outline**: Border only.

## Logo Usage

- Replace the "Moon" icon with the "Date Tree" icon in the header.
- Use the Date Tree as the favicon and app icon.
